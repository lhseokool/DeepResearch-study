# DeepResearch DeepAgent 프로젝트 개요

## 📌 프로젝트 소개

**DeepResearch DeepAgent**는 DeepAgent 프레임워크를 활용하여 구축된 고급 AI 연구 시스템입니다. 이 시스템은 복잡한 연구 질문에 대해 자동으로 정보를 수집하고, 분석하며, 종합적인 보고서를 생성합니다.

## 🎯 주요 목적

1. **자동화된 연구**: 사용자의 질문에 대해 자동으로 웹 검색 및 정보 수집
2. **멀티 에이전트 협업**: 여러 전문 에이전트가 협력하여 연구 수행
3. **지능적 정보 종합**: 수집된 정보를 압축하고 구조화하여 보고서 생성
4. **확장 가능한 아키텍처**: 새로운 도구와 에이전트를 쉽게 추가 가능

## 🔑 핵심 개념

### 1. DeepAgent 프레임워크
- **정의**: 복잡한 멀티 에이전트 시스템을 쉽게 구축할 수 있는 프레임워크
- **특징**: 
  - 서브에이전트 자동 관리
  - 파일시스템 백엔드 지원
  - 미들웨어 기반 확장성

### 2. 서브에이전트 (SubAgent)
각 서브에이전트는 특정 역할을 담당:

- **Researcher**: 웹 검색을 통한 정보 수집
- **Compressor**: 수집된 정보를 압축 및 정리
- **Critic** (선택적): 보고서 품질 검증

### 3. 상태 관리 (State Management)
- **AgentState**: 전체 시스템의 상태 (메시지, 연구 데이터 등)
- **SupervisorState**: 감독자 에이전트의 상태
- **ResearcherState**: 개별 연구자의 상태

### 4. 파일시스템 백엔드
- 연구 결과를 파일로 저장
- 세션 간 데이터 영속성 제공
- `/output/notes/`, `/output/compressed_research.md` 등의 구조화된 디렉토리

## 🚀 작동 방식 (간단 버전)

1. **사용자 질문 입력**: "AI의 최신 동향은?"
2. **오케스트레이터 분석**: 질문을 여러 하위 주제로 분해
3. **Researcher 실행**: 각 하위 주제에 대해 웹 검색 수행
4. **결과 저장**: 수집된 정보를 파일로 저장
5. **Compressor 실행**: 모든 정보를 읽고 압축/정리
6. **최종 보고서 생성**: 구조화된 보고서 작성

## 🔧 주요 기능

### 1. 동적 서브에이전트 생성
- 필요에 따라 새로운 연구자 에이전트를 동적으로 생성
- 병렬 처리로 효율성 향상

### 2. 지능적 정보 압축
- 중복 정보 제거
- 핵심 내용 추출
- 구조화된 형식으로 재구성

### 3. 파일 기반 영속성
- 연구 결과를 파일로 저장
- 세션 간 데이터 유지
- 이전 연구 결과 재활용 가능

### 4. 확장 가능한 도구 시스템
- 새로운 검색 API 추가 가능 (Tavily, Anthropic, OpenAI)
- 커스텀 도구 등록 가능
- MCP (Model Context Protocol) 지원
