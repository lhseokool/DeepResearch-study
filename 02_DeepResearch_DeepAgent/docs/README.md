## 📖 문서별 상세 설명

### 00_프로젝트_개요.md
**대상**: 프로젝트를 처음 접하는 사람  
**내용**:
- 프로젝트 소개 및 목적
- 핵심 아키텍처 개요
- 주요 개념 설명
- 프로젝트 구조
- 작동 방식 (간단 버전)
- 핵심 기능 소개

**읽는 데 걸리는 시간**: 약 10분

---

### 01_핵심_파일_설명.md
**대상**: 코드 구조를 이해하고 싶은 사람  
**내용**:
- 파일 구조 개요
- 각 파일의 역할과 책임
- 주요 클래스 및 함수 설명
- 파일 간 관계도
- 코드 예시

**다루는 파일**:
- `configuration.py` - 설정 관리
- `state.py` - 상태 정의
- `separate_agent.py` - 메인 에이전트
- `utils.py` - 유틸리티
- `prompts/` - 프롬프트 템플릿
- `subagents/` - 서브에이전트 정의
- `skills/` - 스킬 레지스트리
- `tools/` - 커스텀 도구

**읽는 데 걸리는 시간**: 약 20분

---

### 02_상태_관리_상세.md
**대상**: 상태 관리를 깊이 이해하고 싶은 사람  
**내용**:
- 상태(State)의 개념과 중요성
- 상태 구조 계층
- 각 상태 클래스 상세 설명
  - `AgentState`
  - `SupervisorState`
  - `ResearcherState`
- 구조화된 출력 모델
- 리듀서 함수 동작 원리
- 상태 흐름 예시
- 디버깅 팁

**읽는 데 걸리는 시간**: 약 25분

---

### 03_서브에이전트_동작원리.md
**대상**: 서브에이전트의 동작을 이해하고 싶은 사람  
**내용**:
- 서브에이전트 개념
- 각 서브에이전트 상세 설명
  - **Researcher**: 정보 수집
  - **Compressor**: 정보 압축
  - **Critic**: 품질 검증
  - **Dynamic**: 동적 생성
- 서브에이전트 간 협업
- 생명주기 (생성 → 실행 → 종료)
- 실전 팁

**읽는 데 걸리는 시간**: 약 30분

---

### 04_전체_워크플로우.md
**대상**: 전체 시스템 흐름을 이해하고 싶은 사람  
**내용**:
- 워크플로우 개념
- 전체 흐름 한눈에 보기
- 각 단계(STAGE) 상세 설명
  - STAGE 1: 명확화
  - STAGE 2: 연구 계획
  - STAGE 3: 병렬 연구
  - STAGE 4: 정보 압축
  - STAGE 5: 품질 검증
  - STAGE 6: 최종 보고서
- 전체 워크플로우 코드
- 시간 흐름도

**읽는 데 걸리는 시간**: 약 35분

---

### 05_실행_가이드.md
**대상**: 실제로 시스템을 실행하고 싶은 사람  
**내용**:
- 환경 설정 (Python, 패키지)
- API 키 설정
- 기본 실행 방법
- 고급 설정 옵션
- 파일시스템 백엔드 활용
- 실행 옵션 (명확화, 품질 검증 등)
- 디버깅 방법
- 성능 모니터링
- 문제 해결
- 실전 예제

**읽는 데 걸리는 시간**: 약 40분
