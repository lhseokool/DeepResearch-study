# 실행 예시

## 기본 실행

```python
import asyncio
from deep_researcher import deep_researcher

question = "OpenAI와 Anthropic의 AI 안전성 접근법 비교"
result = asyncio.run(
    deep_researcher.ainvoke({
        "messages": {"role": "user", "content": question}
    })
)

print(result["final_report"])
```

## 설정 커스터마이징

```python
result = await deep_researcher.ainvoke(
    {"messages": {"role": "user", "content": question}},
    config={
        "configurable": {
            "research_model": "anthropic:claude-3-5-sonnet-20241022",
            "compression_model": "openai:gpt-4.1-mini",
            "final_report_model": "openai:gpt-4.1",
            "max_concurrent_research_units": 3,
            "max_researcher_iterations": 8,
            "max_react_tool_calls": 5,
            "search_api": "tavily",
            "allow_clarification": True,
        }
    }
)
```

## 실행 흐름 예시

### 입력
```
"2025년 AI 트렌드에 대해 연구해줘"
```

```json

```


### 3. research_supervisor
**Supervisor 계획:**
